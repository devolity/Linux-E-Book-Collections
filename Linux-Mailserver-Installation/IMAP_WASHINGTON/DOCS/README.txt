----------------------------------------------------------------------------
                  University of Washington e-mail tools
----------------------------------------------------------------------------

NOTE!!!!  All Pine files can now be found in the /pine directory
          of this server, also accessible as /mail/PINE.
          A courtesy link for pine.tar.Z remains in /mail.

----------------------------------------------------------------------------

FILES IN THE /mail DIRECTORY:
  README           - This file.
  ccmd.tar.Z       - Command parser from Columbia University used with `ms' 
                     demo client in imap.tar.Z
  EasyMail.tar.Z   - EasyMail source distribution for latest version
  imap.tar.Z       - IMAP developer's kits; includes sources for imapd 
                     server, `ms' demo client, pop2 and pop3 servers, 
                     and c-client library.
  imap-utils.tar.Z - IMAP utilities (imapcopy, mbxcopy, mbxcvt, tmail)
  MailManager.tar.Z - MailManager source distribution for latest version
  pine.tar.Z       - Pine/Pico/IMAPd source distribution for latest version

FILES IN THE /mail/imap-wg DIRECTORY:
  Miscellaneous files relating to the IETF IMAP Working Group for the
  standardization of the Internet Message Access Protocol.

FILES IN THE /mail/interlisp DIRECTORY:
  mm-d.tar.Z       -  MM-D (Xerox Lisp machine IMAP client) source
                      distribution.

FILES IN THE /mail/mac DIRECTORY:
  MacMS            - Original Mac IMAP client from Stanford University.
  Mailstrom        - Newer Mac IMAP client from Stanford University.

FILES IN THE /mail/mime DIRECTORY:
  multimedia-demo.mbox  - MIME torture test mailbox
  <plus other mime-related files>

FILES IN THE /mail/next DIRECTORY:
  EasyMail.app.tar.Z    - Easy to use IMAP client for NeXTs
  MailManager.app.tar.Z - Power user IMAP client for NeXTs

FILES IN THE /mail/sendmail DIRECTORY:
  Modified sendmail 5.65 sources and sample config files
  designed with washington.edu in mind, but usable elsewhere.
  The sources are used almost exclusively on Ultrix systems.

FILES IN THE /mail/TOPS-20 DIRECTORY:
  mapser.tar.Z     -  TOPS-20 IMAP2 server source distribution.
